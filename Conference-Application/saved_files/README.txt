Welcome to the conference! \n
Simply Click run to start the conference. \n
You should see our login Menu, Everyone needs to signup \n
in order to participate in the conference.\n
There are 4 types of user you can choose from: \n
Organizer, Attendee, Speaker and VIPUser. Please follow \n
the Signup instructions. \n
After you login, no matter what type of user you are, \n
you can always send, receive message to each other. \n
Organizers have to use the "Set Conference" option for \n
them to create events, and for Other attendees to signup. \n
You can only use this option once, be careful! \n
You can find the rest of the functions in your user menu, \n
each button has its detailed instruction for you to follow.\n
If you wish to exit the program, please click the "exit" button\n
in the login menu, All OF your progress will be saved.
